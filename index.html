Oração coordenadas

O que são orações coordenadas?

As orações coordenadas são orações independentes que já possuem sozinhas um significado completo.

Oração subordinada

O que são orações subordinadas?

São orações dependentes. Assim, quando separadas, não possuem um sentido completo e, por isso, recebem esse nome, de forma que uma está subordinada à outra.

Eufemismo

Qual a diferença entre eufemismo e litote?

Eufemismo se entende por expressões que amenizam uma coisa ruim. "Ele foi dessa para melhor(Morreu)".

Litote

Qual a diferença entre litote e eufemismo?

A litote serve para negar uma ideia, para dizer o contrário. "Ela não é tão burra(Não é tão inteligente também)"
